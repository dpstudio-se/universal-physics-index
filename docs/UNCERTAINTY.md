# Uncertainty

Measured and estimated quantities may record nominal value, unit, standard uncertainty, confidence interval, uncertainty type, method, sample size and source. Symbolic values should not be given fabricated experimental uncertainty.

For `m = hf/c²`, SI defines `h` and `c` exactly. If frequency is the only uncertain input, standard uncertainty propagates linearly as `u(m) = h u(f) / c²`. More complex and correlated inputs require an explicit propagation model.
