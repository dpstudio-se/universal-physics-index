# Provenance

Scientific records should identify authorship and source using DOI, arXiv ID, URL, book citation, dataset ID, laboratory note, Git commit, publication/retrieval dates, page reference and source hash as applicable.

`source_status` distinguishes `peer_reviewed`, `preprint`, `dataset`, `textbook`, `standard`, `official_document`, `personal_note` and `unknown`. An arXiv identifier never implies peer review. Hashes support integrity checks, not scientific endorsement.
