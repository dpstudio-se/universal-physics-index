# Falsification-First Records

A `HYP` record needs at least a test method, predicted observation, falsification condition, required dataset or measurable variable. Claims awaiting future instruments may use `testability = future`; otherwise untestable boundaries should be `STOP` or conceptual claims `SYM`.

A claim that 8 Hz organizes a domain must define the domain and “organizes,” null model, control frequencies, instrument, statistical threshold, replication rule and confounders before scientific promotion.
