# Status Model

- `EST`: accepted within the stated scientific domain and supported by provenance.
- `DER`: follows mathematically from explicit assumptions; not automatically a new law.
- `HYP`: falsifiable, unverified claim with test metadata.
- `STOP`: unresolved boundary, missing mechanism or presently future-testable claim.
- `ERR`: invalid, dimensionally inconsistent, rejected or unsupported promotion.
- `SYM`: conceptual, philosophical, mythological or non-physical mapping.

Promotion requires evidence and review. Elegance, repeated numbers, agent agreement, visual similarity or a simulation's shape are insufficient. `m = hf/c²` is a rearranged energy-equivalence relation under compatible assumptions, not a new law.
