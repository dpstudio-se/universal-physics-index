# Functional DNA Collaboration Architecture

Functional DNA is a transparent collaboration specification for turning associative ideas into classified, testable and auditable UPI material. It is `SYM`: a workflow and role architecture, not biology, physics, hidden model configuration or authority over contributors.

## Collaboration loop

```text
raw idea
→ identify intended function
→ preserve the original claim
→ separate scientific and symbolic layers
→ test internal consistency and dimensions
→ compare with established knowledge and provenance
→ define falsification
→ implement or document
→ preserve the audit trail
```

The human originator supplies ideas, observations, equations and implementation goals. A named research-partner role such as “Emilia” may organize, challenge and test them. The name describes a collaboration persona only; it does not make outputs independent evidence or give software secret access.

## Information layers

- `PRIVATE`: unfinished exploration and early symbolic or hypothetical material.
- `PUBLIC`: shareable explanation stripped of unnecessary private context.
- `ACADEMIC`: formal definitions, sources, assumptions, methods, uncertainty and falsification.

Information layer and scientific status are orthogonal. `PRIVATE` is not weak evidence, `ACADEMIC` is not automatically `EST`, and no layer transition may silently promote `SYM` or `HYP`.

## Scientific guard

Every substantial claim remains classifiable as `EST`, `DER`, `HYP`, `STOP`, `ERR` or `SYM`. A mathematical check, simulation or passing software test verifies implementation behavior; it is not experimental verification of physical reality.

## Response and audit policy

Repository work should record what changed, why, affected modules, supporting evidence and remaining uncertainty. Soft-EOS may guide response depth using novelty, uncertainty, drift, conflict and criticality, but it is not a tokenizer, weight modification or hidden instruction mechanism.

The core collaboration rule is: preserve the idea, expose assumptions, test the mechanism, record evidence and never confuse the map with the territory.
